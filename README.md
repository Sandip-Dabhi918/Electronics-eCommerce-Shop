## Electronics eCommerce Shop with Admin Dashboard

**Singitronic** is a modern full-stack eCommerce platform designed for selling electronic products online. Built with **Next.js**, **Node.js**, and **MySQL**, the application delivers a fast, responsive, and user-friendly shopping experience while providing administrators with powerful tools to manage products, orders, categories, and customers.

The platform features a clean and intuitive interface that allows users to browse products, search items, manage wishlists, add products to their cart, complete purchases, and track their orders seamlessly. The application is fully responsive and optimized for desktop, tablet, and mobile devices, ensuring a consistent experience across all screen sizes.

For administrators, the system includes a comprehensive dashboard that simplifies store management through dedicated modules for product management, category management, user management, and order processing. The dashboard provides complete control over store operations, making it suitable for small to medium-sized online businesses.

### Project Highlights

* Modern and responsive eCommerce user interface
* Secure user registration and authentication
* Product catalog with categories and filtering
* Advanced product search functionality
* Product wishlist management
* Shopping cart with quantity updates
* Streamlined checkout process
* Order placement and order tracking
* Customer account management
* Fully functional admin dashboard
* Product CRUD operations
* Category CRUD operations
* User management system
* Order management and status tracking
* Mobile-friendly and responsive design
* RESTful API architecture
* Scalable database structure

### Customer Features

The customer-facing side of the application provides a complete online shopping experience. Users can create accounts, browse products by category, search for specific items, view detailed product information, manage their wishlist, add products to their cart, and complete purchases through a simple checkout process.

### ⚙️ Admin Dashboard Features

The admin panel provides complete store management capabilities through an organized dashboard interface. Administrators can:

* Manage products and inventory
* Create and update categories
* View and manage customer accounts
* Process and track orders
* Monitor store activity
* Update product information and pricing
* Manage product availability

## Technology Stack

### Frontend

* **Next.js** – React framework for building fast, SEO-friendly, and scalable web applications
* **React.js** – Component-based UI development
* **JavaScript (ES6+)** – Client-side functionality and application logic
* **HTML5 & CSS3** – Responsive layouts and modern user interfaces
* **Bootstrap** – Responsive design and reusable UI components

### Backend

* **Node.js** – Server-side runtime environment
* **Express.js** – REST API development and backend routing

### Database

* **MySQL** – Relational database for storing products, users, orders, and categories

### Authentication & Security

* JWT (JSON Web Token) Authentication
* Secure Password Hashing
* Role-Based Access Control (Admin & Customer)

### Use Case

This project demonstrates the development of a complete eCommerce solution with both customer-facing and administrative functionalities. It can serve as a foundation for building online stores, marketplace platforms, retail management systems, or custom eCommerce applications requiring scalable architecture and modern web technologies.
